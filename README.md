# BCGDV
BCG DV Internship Application
As a part of application process, I was asked to generate an API key and submit a POST request with my name and email address. I created a new environment called 'BCG DV Internship' and added two variables 'url' and 'api_key'. I generated an API Key for the URL provided using GET request and submitted my name and email using that API Key through POST request.
